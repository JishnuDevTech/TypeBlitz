# TypeBlitz 🎮⚡

TypeBlitz is a fast-paced browser typing game built using HTML, CSS, and JavaScript.

Words float upward like clouds, and the player must type them correctly before they disappear. The game increases in intensity with different difficulty modes and includes real-time scoring, countdown timer, and animated burst effects.

---

## 🚀 Features

- 🎯 Three difficulty modes (Easy, Medium, Hard)
- ⏳ Countdown timer with last 10-second pressure effect
- 💥 Burst animation on correct word
- ❌ Miss penalty system
- 📊 Live score tracking
- ⚡ WPM (Words Per Minute) calculation
- 🧠 Coding-themed word sets

---

## 🕹️ How to Play

1. Select difficulty.
2. Click **Start Game**.
3. Type the floating words exactly before they escape.
4. Survive the timer and score as high as possible.
